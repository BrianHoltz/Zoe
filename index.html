Zoe

Zoe: a 2-D universe where organisms hunt, mate, and reproduce under the control of programs that mutate and evolve. Each Zoe organism ("bug") is a simulated creature that can
  • feel and see nearby bugs,
  • turn and move toward or away from sensed bugs,
  • get energy from sunlight or from bugs it bites,
  • sense physical and genetic attributes of itself and of nearby bugs,
  • control its behavior with a program running on its virtual machine, and
  • clone and mate and evolve through mutation and sexual recombination.
For more information, see the documentation.

To run Zoe:
  1. download Zoe.zip
  2. double-click Zoe.jar inside the unzipped folder
  3. The first time you run Zoe.jar on a Mac, you will have to approve the app in System Preferences -> Security and Privacy
  4. World -> Load Random Bugs, or World -> Load Predefined Bugs to load bugs from the bugs subfolder.
  5. Run -> Go
